# React-Node.js-MySQL-Blog-App

This project is a blog application developed using React, Node.js, MySQL, Quill, and JSON Web Tokens (JWT).

**This project is still under development.**

## Technologies

**Frontend:**
- React
- React Router DOM
- Axios
- Quill
- Vite
- SASS

**Backend:**
- Node.js
- Express.js
- MySQL
- bcryptjs
- JSON Web Tokens (JWT)

### API (Backend)
```bash
cd api
npm install
npm run dev
```

### Client (Frontend)
```bash
cd client
npm install
npm run dev
```
